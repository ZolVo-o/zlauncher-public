# zollauncher Public

Публичный репозиторий проекта zollauncher.

## Что внутри

- Desktop лаунчер (Electron + React)
- Сайт проекта
- Мод доступности для Minecraft (Fabric 1.21.1)

## Быстрый старт

- Сборка desktop: `npm run desktop:build`
- Сборка сайта: `npm run build:site`
- Dev режим: `npm run dev`
