# ZLauncher Access (Minecraft Mod)

Клиентский мод доступности для Minecraft `1.21.1` (Fabric).

## Что делает

- Включает спецрежим для игры одной рукой.
- Быстро переключает сторону управления (правая/левая).
- Добавляет тумблеры приседания и спринта.

## Клавиши по умолчанию

- `F8` - включить/выключить спецрежим
- `F9` - переключить сторону
- `Right Alt` - тумблер приседания
- `\` - тумблер спринта

## Правый режим управления

При включенном режиме и выбранной правой стороне:

- `I/J/K/L` - движение
- `O` - прыжок
- `U` - приседание
- `P` - спринт
- `H` - инвентарь
- `Enter` - атака
- `Right Shift` - использовать предмет

## Сборка

1. Установи `JDK 21`.
2. Перейди в папку мода:
   `minecraft-mod/zlauncher-access`
3. Выполни сборку:
   `gradle build`

После сборки `.jar` будет в:
`minecraft-mod/zlauncher-access/build/libs/`

## Установка

1. Установи Fabric Loader для Minecraft `1.21.1`.
2. Положи собранный `.jar` в папку `mods`.
3. Запусти игру с Fabric профилем.

## Конфиг

Файл конфигурации создается автоматически:
`config/zlauncher-access.json`

## Важно

При выключении спецрежима мод возвращает стандартные клавиши Minecraft по умолчанию.
